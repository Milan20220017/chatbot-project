Overview
BotSpoof is a simple AI-powered chatbot application that allows users to interact with a virtual assistant in real-time. Users can send messages and receive responses based on predefined rules and training data.
The project includes a modern frontend built with React/Preact and a backend powered by Node.js, Express, and MongoDB.
Features
Real-time chat interface with message input and bot responses.
Simple AI logic with keyword-based responses (training dataset included).
Loading indicator when the bot is generating a response.
Styled with TailwindCSS for responsive and modern UI.
User-friendly interface with message bubbles for user and bot.
Cross-platform: can run on any device with Node.js and a browser.
Technologies Used

Frontend:
React (JSX) / Preact
TailwindCSS for styling
react-icons for UI icons
Backend:
Node.js
Express.js
Axios (for HTTP requests from frontend)
MongoDB (database for storing messages if needed)

Build Tool:
Vite (for frontend development)
